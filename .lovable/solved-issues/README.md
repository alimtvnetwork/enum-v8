# Solved Issues

This folder will hold issues moved from `.lovable/pending-issues/` once they are resolved. Each file must include `## Solution`, `## Iteration Count`, `## Learning`, and `## What NOT to Repeat` sections.

Currently empty — the active blockers in `pending-issues/` (#01 = Task W upstream rename, #02 = cross-spec stale paths) are still in flight.

## Conventions

When moving an issue here:
1. Move (not copy) the file from `pending-issues/` to `solved-issues/` with the same numeric prefix.
2. Append the four required sections to the bottom.
3. If the resolution revealed a forever-forbidden pattern, also add an entry to `.lovable/strictly-avoid.md`.
4. Update `.lovable/plan.md` to reflect the closure.
